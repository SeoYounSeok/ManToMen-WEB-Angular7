# INTRO
 
This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.0.2.

### 1) Angular 사전준비
  1. Angular CLI 설치
  2. Build

### 1) Angular CLI 설치
- 프로젝트와 콤포넌트 생성에 이용됩니다.

	```
	npm install -g @angular/cli
	```
	```
	ng --version
	```
	

### 2) Build
- 배포를 하고싶다면 빌드를 해줘야한다. dist 라는 폴더가 생성되는 것을 확인할 수 있다.

	```
	ng build
  ```
